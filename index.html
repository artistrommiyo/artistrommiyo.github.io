My Projects

Below are some of the key projects I have developed, utilizing modern technologies to deliver scalable, interactive, and user-friendly applications.

Omnicare - Customer Support System

Technologies Used: HTML, CSS, TypeScript, Angular, Leaflet, RESTful APIs

Developed a customer support application to optimize communication between customers and support representatives. The application features ticket management, live chat, and a knowledge base. It also includes analytics for performance tracking. Leveraged Angular to create dynamic components and Leaflet for integrating the interactive map. The system also includes a role-based access control feature for improved security.

1&1 Omnidesk GIS - Customer Support with GIS Mapping

Technologies Used: HTML, CSS, TypeScript, Angular, Leaflet, RESTful APIs

Built a customer support application integrated with GIS mapping functionality to provide location-based insights. This application visualizes customer data and service requests on an interactive map using Leaflet. The system pulls real-time data via RESTful APIs, enabling users to view active service requests and customer locations on the map. Angular was used for building interactive components like ticket management, and a role-based access control system was implemented to ensure secure data handling.
